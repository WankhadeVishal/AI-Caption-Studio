Introduction

The AI-Powered Video Caption Generator is a full-stack web application designed to automate the process of subtitle generation for videos. The system allows users to upload an MP4 video, extract its audio, transcribe spoken content using OpenAI Whisper, optionally translate subtitle text into another language, generate subtitle files in SRT and VTT formats, and burn subtitles into the final video using FFmpeg. The application is built with a modern Streamlit interface to make the overall workflow simple, interactive, and user-friendly.

With the rapid growth of video-based learning, digital marketing, and content creation, subtitles have become essential for accessibility, engagement, and multilingual reach. Manual subtitle creation is time-consuming and inefficient. This project addresses that challenge by delivering an automated solution that combines artificial intelligence and multimedia processing into one integrated platform.

Problem Statement

Manual subtitle creation requires considerable time, effort, and technical skill. Many creators, students, and small teams need a lightweight system that can automatically generate, translate, and embed subtitles into videos without depending on expensive commercial tools. Therefore, there is a clear need for a user-friendly application that simplifies the end-to-end caption generation process.

Objectives

  • To build a web application that accepts MP4 video uploads.
  • To extract audio from uploaded videos automatically.
  • To transcribe speech into text using OpenAI Whisper.
  • To generate subtitle files in SRT and VTT formats.
  • To optionally translate subtitles into another language using the OpenAI API.
  • To burn subtitles into the final video using FFmpeg.
  • To provide a modern user interface with previews, progress tracking, and download support.

Scope of the Project

This project focuses on local video caption generation for desktop use. It supports MP4 uploads, audio extraction, Whisper transcription, subtitle export, optional translation, and subtitle burn-in. The system is suitable for educational use, small-scale creator workflows, academic mini-projects, and demonstrations of AI-powered multimedia automation.

Proposed Solution

The proposed solution is a modular Python application with a Streamlit frontend and utility-based backend. The user uploads a video through the interface, after which the application extracts audio in WAV format. The extracted audio is passed to Whisper for transcription. The resulting transcript is converted into timestamped subtitle segments and exported as SRT and VTT files. If the user selects a different subtitle language, the subtitle text is translated using the OpenAI API. Finally, FFmpeg burns the subtitle file into the original video and produces a captioned MP4 file for download.

System Modules

1. User Interface Module

Handles file upload, language selection, processing controls, progress feedback, previews, and downloads.

2. Audio Extraction Module

Extracts speech audio from the uploaded video and converts it into WAV format suitable for transcription.

3. Transcription Module

Uses OpenAI Whisper to convert audio into timestamped text segments.

4. Subtitle Generation Module

Converts transcript segments into standard subtitle formats such as SRT and VTT.

5. Translation Module

Translates subtitle text into another language using the OpenAI API when required.

6. Subtitle Burn-In Module

Uses FFmpeg to embed subtitles directly into the exported MP4 file.

Working Methodology

  1. User uploads an MP4 video.
  2. The system saves the video in the uploads directory.
  3. Audio is extracted and converted into WAV format.
  4. Whisper transcribes the spoken content into timestamped segments.
  5. SRT and VTT subtitle files are generated.
  6. If required, subtitles are translated into the selected language.
  7. FFmpeg burns subtitles into the video.
  8. The final outputs are displayed and made available for download.
